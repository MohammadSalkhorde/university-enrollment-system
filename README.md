# سامانه جامع انتخاب واحد دانشگاهی
این پروژه بر اساس اصول SOLID و متدولوژی Agile طراحی شده است.

### ویژگی‌های کلیدی:
- احراز هویت هوشمند بر اساس نقش (مدیر، استاد، دانشجو)
- امنیت پیشرفته با JWT
- منطق انتخاب واحد (چک کردن تداخل، پیش‌نیاز و ظرفیت)
- رابط کاربری کاملاً فارسی و راست‌چین (RTL)

### نحوه اجرا:
1. نصب کتابخانه‌ها: `pip install -r requirements.txt`
2. مهاجرت دیتابیس: `python manage.py migrate`
3. ایجاد مدیر سیستم: `python manage.py createsuperuser`
4. اجرای سرور: `python manage.py runserver`

### تست‌های واحد:
برای اجرای تست‌های کیفیت: `python manage.py test apps.students`
